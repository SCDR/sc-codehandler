## Gradio
切换web-ui分支可使用Gradio的前端配置相关设置【推荐】

## 功能

- 自动整理软著源代码到word文档
- 超过60页截取前后30页

## 运行

- 运行start.sh安装依赖并运行，在浏览器打开目标地址

## 配置
- 在浏览器中进行相关配置，处理后下载目标文件
