## 功能

- 自动整理软著源代码到word文档
- 超过60页截取前后30页

## 运行

- 安装依赖后配置config.json中的源代码目录codeFile.directory和项目名称info.name
- codeFile.extensions配置文件后缀
- 入口为src/run.py
